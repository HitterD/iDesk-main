Klik item Installation → kemana?

Saat admin klik baris di tab Installation, halaman mana yang dibuka?

A

Ke Detail Hardware Request

Klik baris → buka /hardware-requests/:id dengan section instalasi ter-highlight. Admin melihat konteks lengkap: procurement + instalasi dalam 1 halaman.

Flow: Tab Installation → HW Request Detail
Tab Installation
→
Hardware Request Detail
✅ Lihat request asal + item + status procurement
✅ Lihat jadwal instalasi + assignment
✅ 1 halaman untuk semua konteks
B

Ke Detail Ticket

Klik baris → buka /tickets/:id (halaman ticket detail yang sudah ada). Admin bisa update status, assign, chat, dll. Lalu ada link balik ke hardware request.

Flow: Tab Installation → Ticket Detail
Tab Installation
→
Ticket Detail
↔
HW Request (link)
✅ Bisa update status, assign, messaging
✅ Familiar — UI ticket detail sudah ada
⚠️ Perlu navigasi tambahan untuk lihat konteks procurement
