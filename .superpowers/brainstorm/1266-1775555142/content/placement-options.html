Di mana Hardware Installation ditampilkan?

Saat ini Hardware Installation tickets tercampur di halaman Tickets umum. Pilih cara terbaik untuk mengintegrasikannya ke halaman Hardware Requests.

A

Tab Baru "Installation"

Tambah tab Installation di samping tab Pending / Purchasing / Arrived / Completed pada halaman Hardware Requests.

Hardware Requests - Tab Bar
All 5 Pending 1 Purchasing 2 Arrived 1 Completed 1 🔧 Installation 3
Daftar hardware installation tickets muncul di sini

Pros

  • Mudah ditemukan, 1 klik akses
  • Konsisten dengan UI tab yang sudah ada
  • Terpisah jelas dari procurement flow

Cons

  • Tab bar makin panjang
  • Terpisah dari konteks request asal
B

Inline dalam Card Request

Instalasi ditampilkan sebagai bagian dari setiap Hardware Request card — muncul ketika status item sudah Arrived.

Hardware Request Card
PC AIO - Dept. Finance Arrived
🔧 INSTALLATION
Scheduled: 10 Apr 2026, 08:00-12:00
Status: In Progress

Pros

  • Konteks lengkap — lihat request + instalasi sekaligus
  • Tidak perlu navigasi tambahan

Cons

  • Sulit melihat semua instalasi sekaligus
  • Admin harus scroll satu-satu
C

Tab + Inline (Hybrid)

Gabungan: Tab "Installation" untuk overview semua instalasi, dan info instalasi juga muncul inline di card request masing-masing.

Best of Both Worlds
All Pending Purchasing Arrived Completed 🔧 Installation
Tab → overview semua instalasi
Card → detail instalasi per request

Pros

  • Overview + detail kontekstual
  • Admin bisa kerja dari kedua arah

Cons

  • Lebih kompleks untuk dibangun
  • Info duplikat di 2 tempat
